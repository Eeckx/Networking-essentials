‚öôÔ∏è Security Hardening Stappen

  1. Server Role & Feature Minimalisatie: installeer alleen benodigde roles/features om attack surface te verkleinen.
  2. Windows Firewall: configureer inbound/outbound regels, beperk poorten tot wat noodzakelijk is.
  3. User Account Hardening:
    • Verwijder standaard accounts of wijzig standaardnamen.
    • Gebruik sterke wachtwoorden en MFA waar mogelijk.
    • Voer accounts lockout policies in.
  4. Service & Process Hardening:
    • Stop ongebruikte services.
    • Gebruik AppLocker of Windows Defender Application Control (WDAC) voor applicatiebeperkingen.
  5. Group Policy Hardening:
    • Beveilig domain controllers via GPO.
    • Configureer security baselines via Microsoft Security Compliance Toolkit.
    • Harde wachtwoordpolicy, auditpolicy, account lockout en user rights assignments instellen.
  6. BitLocker & Disk Encryption: bescherm gegevens bij diefstal van hardware.
  7. Secure Network Configuration:
    • Schakel SMBv1 uit.
    • Gebruik TLS/HTTPS voor alle communicatie.
    • Beperk RDP-toegang via firewall en Network Level Authentication (NLA).
  8. Anti-Malware:
    • Windows Defender/ATP inschakelen en real-time protection configureren.
    • Regelmatige malware scans plannen.
  9. Auditing & Logging:
    • Activeer en configureer Security Event Logs.
    • Gebruik Windows Event Forwarding of SIEM integratie voor centrale monitoring.
  10. Patch & Update Management: hou servers up-to-date via WSUS of SCCM.
  11. Backup & Recovery: configureer regelmatige back-ups en test herstelprocedures.
