🗂️ Group Policy Management – Beheer van bedrijfsregels en security instellingen

🎯 Doelstelling

Deze handleiding leert hoe je Group Policy Objects (GPO’s) installeert, configureert en beheert in een Windows Server-omgeving. GPO’s bieden centraal beheer van beveiligingsinstellingen, software deployment, desktopconfiguraties en gebruikersrechten in een Active Directory domein.

🧰 Voorwaarden

  • Windows Server 2019, 2022 of nieuwer
  • Active Directory Domain Services (AD DS) geïnstalleerd
  • Administrator rechten op de Domain Controller
  • Basiskennis van OU’s (Organizational Units) en domeinstructuur

⚙️ Installatie van Group Policy Management Console (GPMC)

  • Open Server Manager → Add Roles and Features → Features → Group Policy Management.
  • PowerShell alternatief: 
    
Install-WindowsFeature GPMC
  • Na installatie is de Group Policy Management Console beschikbaar via Server Manager → Tools → Group Policy Management.

🏗️ GPO maken en koppelen

  1. Open Group Policy Management Console (GPMC).
  2. Navigeer naar je domein → rechterklik op een OU of het domein zelf → Create a GPO in this domain.
  3. Geef het GPO een herkenbare naam (bijv. Security-Baseline).
  4. Klik op Edit om de policy te configureren.
  5. Koppel GPO aan een OU door deze te slepen of via Link an Existing GPO.

⚡ Veelgebruikte GPO-instellingen in bedrijven

  • Security Policies:
    • Password Policies: minimale lengte, complexiteit, verlopen
    • Account Lockout Policies: blokkering na aantal mislukte logins
    • User Rights Assignment: wie mag server beheren, RDP toegang, shutdown rechten
  • Windows Defender & Security:
    • Real-time bescherming inschakelen
    • Controlled Folder Access voor ransomware bescherming
    • Exploit Guard & Attack Surface Reduction rules
  • Software Deployment:
    • MSI-installatie via Computer Configuration → Software Settings
    • Automatische updates voor interne apps
  • Desktop & User Configuration:
    • Mapped drives en printers via Preferences → Drive Maps / Printers
    • Start Menu en Taskbar instellingen
    • Folder redirection: Documents, Desktop naar server shares
  • Remote Desktop & RDS:
    • Session timeouts, reconnection policies
    • Clipboard en drive redirection beperken
  • Network & Browsers:
    • Proxy instellingen
    • Internet Explorer/Edge security zones
    • Wireless network policies
  • Auditing & Logging:
    • Audit account logon events, object access
    • Advanced Security Audit policies

🔧 GPO configuratie voorbeelden

  • Forceer complexe wachtwoorden: 
    
Set-ADDefaultDomainPasswordPolicy -Identity "contoso.com" -ComplexityEnabled $true -MinPasswordLength 12
  • Verberg specifieke drives op desktops via GPO: 
    
Computer Configuration\Policies\Administrative Templates\Windows Components\File Explorer\Hide these specified drives in My Computer
  • Software deployment (MSI): 
    
Computer Configuration\Policies\Software Settings\Software Installation
  • Folder Redirection: 
    
User Configuration\Policies\Windows Settings\Folder Redirection\Documents

🧠 Testing en validatie

  1. Forceer een update van GPO op een client: 
    
gpupdate /force
  2. Controleer toegepaste policies: 
    
gpresult /r
  3. Gebruik Event Viewer → System / Security logs om errors of blocked policies te controleren
  4. Test policies eerst in een test-OU voordat ze breed uitgerold worden

💡 Best Practices

  • Houd GPO’s zo georganiseerd en minimaal mogelijk
  • Gebruik beschrijvende namen voor GPO’s (bijv. HR-Workstations-DriveMap)
  • Documenteer wijzigingen en test eerst in een testomgeving
  • Combineer policies met Security Baselines van Microsoft
  • Controleer periodiek of GPO’s correct worden toegepast met gpresult / Event Viewer
  • Gebruik “Loopback processing” voor RDS servers indien nodig

🧾 Conclusie

Group Policy Management is een krachtige manier om centrale controle en standaardisatie te realiseren binnen een Windows-omgeving. Door zorgvuldig beleid te ontwerpen, testen en implementeren kun je zowel security, configuratie als software deployment consistent beheren over alle servers en clients.
