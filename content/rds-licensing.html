ğŸ§° Kernconcepten

  • RDS Licensing: mechanismen waarmee Windows Server licenties beheert voor Remote Desktop Services.
  • Licensing Server: server die RDS CALs (Client Access Licenses) beheert en toewijst.
  • RDS CAL: licentie die een gebruiker of device toestaat om een RDS-sessie te starten.
  • Licensing Modus: bepaalt hoe licenties worden toegewezen: per gebruiker of per apparaat.
  • Compliance: voldoende CALs aanwezig voor alle gebruikers/devices; controleren via RD Licensing Manager.

âš™ï¸ RDS Licensing Modi

  • Per Device (Per apparaat):
    • Elke fysieke of virtuele machine die een RDS-sessie start, vereist een CAL.
    • Geschikt voor gedeelde werkplekken, labomgevingen of kiosk devices.
    • Licenties blijven toegewezen aan het device, onafhankelijk van gebruiker.
  • Per User (Per gebruiker):
    • Elke gebruiker die toegang heeft tot RDS-sessies, vereist een CAL.
    • Gebruiker kan vanaf meerdere devices inloggen met dezelfde CAL.
    • Geschikt voor werknemers met meerdere devices (laptop, desktop, tablet).

ğŸ“Š Best Practices

  • Kies de licentiemodus op basis van gebruikerspatroon: Per User voor meerdere devices per gebruiker, Per Device voor gedeelde terminals.
  • Documenteer alle toegewezen CALs en controleer compliance periodiek.
  • Zorg dat de licentieserver altijd online en bereikbaar is door RD Session Host servers.
