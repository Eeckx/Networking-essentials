üõ°Ô∏è Configuratie van Windows Defender Antivirus

  • Real-time bescherming inschakelen (verplicht voor continue bescherming)
  • Cloud-delivery en automatische sample submission voor nieuwste detecties:

Set-MpPreference -SubmitSamplesConsent SendSafeSamples
  • Exclusies instellen voor kritieke toepassingen of storage (indien nodig):

Add-MpPreference -ExclusionProcess "vmms.exe"
Add-MpPreference -ExclusionExtension ".vhdx"
  • Periodieke scans plannen via Task Scheduler of via PowerShell: 
    
Set-MpPreference -ScanScheduleDay 0 -ScanScheduleTime 03:00
    (0 = zondag, 1 = maandag, etc.)

üíæ Updates & Signatures

  • Handmatige update via PowerShell:

  • Controleer laatste update:

  • Integratie met WSUS mogelijk voor interne update-distributie

üß∞ Security Policies & Hardening

  • Configureer via **Group Policy** of **Local Security Policy**:
    • Exploit Guard (Attack Surface Reduction)
    • Controlled Folder Access (bescherming tegen ransomware)
    • Network Protection en SmartScreen
    • Credential Guard en Device Guard (indien Hyper-V geactiveerd)
  • PowerShell configuratie voorbeelden: 
    
Set-MpPreference -AttackSurfaceReductionOnlyExclusions @()
Set-MpPreference -EnableControlledFolderAccess Enabled
  • Audit policies inschakelen via GPO om verdachte activiteit te loggen

üí° Best Practices
